# Restaurant Management System (Java + SQLite + Apache POI)

A full-featured Restaurant Billing and Management desktop application built with **Java (Swing)**, **SQLite**, and **Apache POI**. This system allows restaurant staff to manage dishes, take orders, generate bills, and export data to Excel for reporting.

---

## Features

- **Add/View/Delete Dishes**  
- **Billing System** with auto-calculated totals  
- **Export to Excel**:  
  - Export `Dishes` data from `restaurant.db`  
  - Export `Bills` data from `bills.db`
- Local database storage using **SQLite**
- Clean UI built using **Java Swing (NetBeans Designer)**
